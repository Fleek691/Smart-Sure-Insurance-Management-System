# SmartSure Angular Frontend

This app follows the architecture document: Angular 17, gateway-only API access, OTP registration, customer policy/claims screens, admin operations, and report views.

## Run

```bash
npm install
npm start
```

## Backend integration

- Gateway base URL: `http://localhost:5000/gateway`
- Auth routes: `/auth/*`
- Policy routes: `/policies/*`
- Claims routes: `/claims/*`
- Admin routes: `/admin/*`
